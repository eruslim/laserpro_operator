# LaserCut Pro - Operator Portal

Production floor interface for managing assigned jobs and updating production status.

**Portal**: Operator Portal  
**Port**: 5175  
**Role Required**: `operator`  
**Status**: Production Ready ✅

## Quick Start

```bash
npm install
npm run dev
```

Visit: **http://localhost:5175**

## Features

- View assigned jobs
- Download design files
- Update production status (cutting → post-processing → quality check → packaging)
- Add production notes
- Track job history
